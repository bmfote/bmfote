Everyone in AI is adding complexity. More agents. More tools. More orchestration layers. More config files. More infrastructure to manage infrastructure.

I've been going the other direction. Single thread instead of 10 templates. SQLite instead of a vector database. Full text search instead of RAG. Hooks instead of a memory framework. Each iteration I remove something and it gets better. Not because the model improved. Because there's less surface area for things to break.

The best AI harness isn't the one with the most features. It's the one with the fewest moving parts that still gets the job done. Every layer you add is a layer that can fail, confuse the model, or drift out of sync. If you can get the same result with less, you should.
